### 문제 설명

게임 캐릭터를 4가지 명령어를 통해 움직이려 합니다. 명령어는 다음과 같습니다.

### 나의 답안

```python
def solution(dirs):
    # 0. 변수
    dr = [-1, 1, 0, 0] # UDRL
    dc = [0, 0, 1, -1]
    
    dir_to_num = {
        'U': 0,
        'D': 1,
        'R': 2,
        'L': 3
    }
    
    # 1. 좌표 지정
    r, c = 5, 5
    
    # 2. 이동하며 기록하기
    visited = set()
    for dir in dirs:
        dir_num = dir_to_num[dir]
        nr = r + dr[dir_num]
        nc = c + dc[dir_num]
        if nr < 0 or nc < 0 or nr > 10 or nc > 10:
            continue
        visited.add((r, c, nr, nc))
        visited.add((nr, nc, r, c))
        r, c = nr, nc
    
    # 3. 숫자 세기
    return len(visited)//2
```

### 생각

구현 방향을 잘 잡아야하는 문제이다. 좌표 평면이라 2차원 배열을 먼저 생각해서 구현하기 쉬우나, 잘 생각해보면 2차원 배열이 필요없다. 2차원 좌표를 사용해서 리스트로 관리하면서 문제를 해결할 수 있다. 나는 중복을 피하는 자료구조인 set을 사용하였다. 길은 양쪽으로 이동할 수 있기때문에 처음 저장시 양쪽을 모두 저장하고 마지막에 반으로 나눠주는 방법을 택했다.
